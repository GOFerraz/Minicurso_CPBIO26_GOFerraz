# Ecossistemas sob uma nova perspectiva: possibilidades de análise com Sistemas de Informação Geográfica e Sensoriamento Remoto
O minicurso tem o objetivo de explicar os fundamentos básicos da Ecologia da Paisagem, exemplificar como o contexto espacial pode ser explorado em pesquisas ambientais e biológicas, e apresentar as principais ferramentas e softwares utilizados para aquisição, processamento e análise de dados espaciais.
